Chabot Space & Science Center Mobile App


GitHub Repo

The Chabot Space & Science Center is a well-known educational space instition in Oakland, California that also serves as a visitor center for NASA Ames Along with a group of individuals at Codify Berkeley, I helped create an interactive, advanced, accessible, and user-friendly mobile app (that is still being improvised) which currently is fully extensible, has a map template, demo page, exhibit page, and navigation system. Near term implementations will include audio support, push notifications, and visitor feedback.


Machine Learning API - Cerbo


GitHub Repo User Stats

Cerbo is a python based library that provides a user-friendly experience to train and learn about ML models. Cerbo offers educators the ultimate level of abstraction to improve approachability in a beginner-friendly programming language. With over 15K downloads, Cerbo is a tool that teaches the fundamental coding skills associated with AI.
