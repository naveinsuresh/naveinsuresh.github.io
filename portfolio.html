Cerbo is a python based library that provides a user-friendly experience to train and learn about ML models. Cerbo offers educators the ultimate level of abstraction to improve approachability in a beginner-friendly programming language. With over 15K downloads, Cerbo is a tool that teaches the fundamental coding skills associated with AI.
